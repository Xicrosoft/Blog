为什么网站使用cookies？

Cookie 有什么作用？

cookie （称为 Internet 或 Web cookie）是用于描述 由 Web 服务器 提供给 Web 浏览器的一类消息的 术语。cookie 的主要目的是识别用户并可能准备定制的网页或为您保存站点登录信息。

当您使用 cookie 进入 网站 时，可能会要求您填写提供个人信息的表格；例如您的姓名、电子邮件地址和兴趣。此信息被打包成 cookie 并发送到您的 Web 浏览器，然后浏览器存储这些信息以供以后使用。下次您访问同一个网站时，您的浏览器会将 cookie 发送到 Web 服务器。每次浏览器从服务器请求一个页面时，该消息就会被发送回服务器。

Web 服务器没有内存，因此您正在访问的托管网站会在您的计算机硬盘上传输浏览器的 cookie 文件，以便该网站可以记住您的身份和偏好。此消息交换允许 Web 服务器使用此信息为您提供定制的网页。因此，例如，您可能会看到一个带有您名字的欢迎页面，而不是只看到一个通用的欢迎页面。

Cookie 的类型

session cookie
也称为临时 cookie，当您关闭 Web 浏览器时会被删除的 cookie。会话 cookie 存储在临时内存中，关闭浏览器后不会保留。会话 cookie 不会从您的计算机收集信息。他们通常会以会话标识的形式存储信息，不会识别用户的个人身份。

持久性 cookie 
也称为永久 cookie，或存储的 cookie，存储在您的硬盘驱动器上直到过期（持久性 cookie 设置有过期日期）或直到您删除 cookie 的 cookie。持久性 cookie 用于收集有关用户的识别信息，例如 Web 冲浪行为或特定网站的用户偏好。

Cookie 存储哪些信息？

在大多数情况下，cookie 将包含一串包含浏览器信息的文本。为了工作，cookie 不需要知道你来自哪里，它只需要记住你的浏览器。一些网站确实使用 cookie 来存储有关您的更多个人信息。但是，只有当您自己向网站提供了该个人信息时，才能这样做。合法网站将对存储在 cookie 中的个人信息进行加密，以防止有权访问您的 cookie 文件夹的另一方未经授权使用。

Cookie 有六个可以传递给它们的参数：

  • cookie 的名称。
  • cookie 的值。
  • cookie 的到期日期——这决定了 cookie 在您的浏览器中保持活跃的时间。
  • cookie 的有效路径——这设置了我们有效的 cookie 的 URL 路径。该路径之外的网页不能使用 cookie。
  • cookie 有效的域。当站点在域中使用多个服务器时，这使得任何服务器上的页面都可以访问 cookie。
  • 需要安全连接——这表明 cookie 只能在安全的服务器条件下使用，例如使用 SSL的站点。

什么是恶意 Cookie？

Cookie 通常不会危及安全性，但恶意 Cookie 的趋势正在增长。这些类型的 cookie 可用于存储和跟踪您的在线活动。监视您的在线活动的 cookie 称为恶意 cookie 或跟踪 cookie。这些是需要注意的不良 cookie，因为它们会随着时间的推移跟踪您和您的浏览习惯，以建立您的兴趣档案。一旦该配置文件包含足够的信息，您的信息很有可能会被出售给广告公司，然后广告公司会使用此配置文件信息以针对您的兴趣投放特定广告。如今，许多 防病毒程序 在扫描系统中的病毒时 都会标记可疑 的间谍软件 或 广告软件cookie。

查看和删除 Cookie

Cookie 由 Web 浏览器存储在您系统的硬盘上，您可以查看它们以了解您访问的哪些网站与您的 Cookie 文件相关联。

例如，如果使用 Internet Explorer，您选择工具，然后选择 Internet 选项。在常规选项卡上，您将看到标题为浏览器历史记录的部分。单击设置，然后选择查看文件。

这将打开一个 Windows 资源管理器窗口，其中列出了您的所有临时 Internet 文件，包括您的 cookie。每个 cookie 都将由一个站点 URL 标识，从而可以轻松确定哪些 cookie 是您信任并希望保留的，哪些是您在访问网站时不记得并会删除的。

要更改 cookie 设置，只需返回工具，然后选择 Internet 选项。在“隐私”选项卡上，您将看到一个滑块，您可以移动该滑块来调整您的浏览器接受 cookie 的级别。低例如阻止没有紧凑隐私政策的第三方 cookie，并限制第三方 cookie 保存可用于在未经您同意的情况下与您联系的信息。中高也会这样做，但也会阻止保存有关您的信息的第一方 cookie。您可以选择的其他隐私选项是接受所有 cookie 或同时阻止所有 cookie。

如果您使用的是 Internet Explorer 以外的浏览器，您可以访问每个浏览器网站上的以下 cookie 页面，了解如何在使用 Firefox、Opera 或 Safari 时管理您的 cookie。

  • Firefox：  Firefox 帮助：Firefox 的 Cookie 选项
  • Opera：Opera 中的安全、隐私和 Cookie
  • Safari：  Safari 帮助管理 cookie

第一方和第三方 Cookie

在浏览器中选择隐私设置时，您会看到两个术语是“第一方 cookie”和“第三方 cookie”。第一方 cookie 是源自（或发送至）您当前正在查看的网站的那些 cookie。这些类型的 cookie 通常包含有关您对该特定网站的偏好的信息。这些 cookie 通常是第三方 cookie，来自（或将发送到）不是您正在访问的网站的网站。例如，如果您所在的网站使用第三方广告，这些第三方广告网站可能会使用 cookie 来跟踪您的网络习惯以用于营销目的。

虽然有些人可能会简单地选择阻止所有 cookie，但如果您这样做，可能会使网上冲浪变得困难。例如，如果您在网上购物，许多已经使用 cookie 实现的电子商务购物车将无法正常工作。如果您删除或禁用该 cookie，您经常访问的可让您个性化内容的网站也不会在您访问时显示您的偏好。

尽管存在一些误解，但大多数 cookie 都是合法文件，不会侵犯您的隐私。一旦您养成了查看与浏览器相关的 cookie 并通过删除恶意 cookie 或尝试不同的浏览器隐私设置自行管理它们的习惯，您仍然可以保留让冲浪变得轻而易举的好 cookie，而保留坏的可能会在您的系统外跟踪您的浏览习惯的 cookie。
