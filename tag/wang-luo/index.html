什么是Hosts

Hosts 是一个没有扩展名的系统文件,作用是将一些常用的网址域名与其对应的IP地址建立一个关联数据库，当用户在浏览器中输入一个网址时，系统会首先自动从 Hosts 文件中寻找对应的 IP 地址，一旦找到，系统会根据你配置的 IP 进行访问，如果没有找到，则系统会将网址提交到 DNS 域名解析服务器进行 IP 地址的解析。
